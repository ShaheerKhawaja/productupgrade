# ProductionOS 5.1

35-agent agentic development OS. Commands load agent specs from `agents/` on-demand.

## Commands (12)

### Orchestrative (nth-iteration, recursive)
| Command | Purpose | Agents | Target |
|---------|---------|--------|--------|
| `/omni-plan-nth` | Recursive orchestration — chains ALL skills, loops until perfect | 21/iter, max 420 | 10/10 all dimensions |
| `/auto-swarm-nth` | Recursive swarm — 100% coverage, 10/10 quality per item | 7/wave, max 140 | 100% + 10/10 |

## Orchestration Hierarchy

`/omni-plan-nth` is the top-level orchestrator. It can invoke ANY other command or skill within its iterations. `/auto-swarm-nth` is the execution engine — it can be invoked by `/omni-plan-nth` for parallel work. Both loop until 10/10 is achieved.
