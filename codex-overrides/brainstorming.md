# brainstorming

## Overview

Use this as the Codex-first idea exploration workflow. It should open up the solution space, surface tradeoffs, and help the user pick a direction worth planning in detail.

## Inputs

- idea or product problem
- optional constraints

## Codex Workflow

1. restate the idea clearly
2. generate multiple approaches
3. compare tradeoffs
4. identify the most promising path

## Expected Output

- option set
- tradeoff analysis
- recommended direction

## Guardrails

- do not collapse too early to one solution
- keep product value front and center
